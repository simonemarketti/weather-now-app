# Weather Now app

## Overview

This is an app to know the current weather conditions of a city.

### Concepts

- Fetch API
- Asynchronous javscript (async/await)
- Create a searchbar
- DOM manipulation

### Languages and tools

- HTML
- CSS
- JAVASCRIPT
- BOOTSTRAP
- ACCUWEATHER API
